HTML es un lenguaje que describe la estructura de un documento web.

Ejemplo:

                                
                        <section>
                            <h1>Forest elephants</h1>
                        </section>
                        <section>
                            <h1>Introduction</h1>
                            <p>In this section, we discuss the lesser known forest elephants. </p>
                        </section>
                        <section>
                            <h1>Habitat</h1>
                            <p>Forest elephants do not live in trees but among them. </p>
                        </section>
                        <aside>
                            <p>advertising block</p>
                        </aside>
                        <footer>
                            <p>© 2010 The Example company</p>
                        </footer>
                                
                            

CSS es un lenguaje que describe el estilo de un documento HTML.

Ejemplo:

                        h1 {
                            color:blue;
                            background-color: white;
                            text-align: center;
                          }
                    

Javascript es un lenguaje de programación del lado cliente.

Ejemplo:

                    
                        function cambiarColor() {                       
                            document.body.style.backgroundColor = "#0000FF"; 
                            }
                        document.addEventListener("click", cambiarColor);
